Genel Bakis

AMD, 2025 yilinda Windows icin ROCm destegini on izleme olarak sunmaya baslamistir. Bu sayede AMD Radeon RX 7000, RX 9000 serisi GPU'lar ve Ryzen AI Max APU'larla PyTorch kullanarak yerel AI islemleri yapabilirsiniz.

Mevcut Durum (Aralik 2025)

  • PyTorch: Windows icin ROCm destekli on izleme surumu mevcut
  • Inference (Model Calistirma): Calisir durumda
  • Fine-Tuning: Sinirli destek, dikkatli yapilandi gerektirir
  • DirectML: Inference icin calisir, fine-tuning destegi henuz yok
AMD'nin Aciklamasi: "PyTorch for AMD on Windows and Linux is now available as a public preview. You can now use native PyTorch for AI inference on AMD Radeon RX 7000 and 9000 series GPUs and select AMD Ryzen AI 300 and AI Max APUs."

AMD Strix Halo (Ryzen AI Max+ 395)

AMD Ryzen AI Max+ 395 "Strix Halo", 128 GB'a kadar paylasimli bellek kapasitesiyle fine-tuning icin benzersiz bir firsat sunmaktadir. GPU'ya 96-112 GB bellek ayrilabilir.

GPU Radeon 8060S (40 RDNA 3.5 CU)
GPU Saat Hizi Maks. 2.9 GHz
Teorik Performans 59.4 FP16/BF16 TFLOPS (tepe)
Gercek Performans ~36.9 TFLOPS (hipBLASLt ile)
Bellek Bant Genisligi ~212 GB/s (DDR5-8000)
GPU'ya Ayrilabilir Bellek 96-112 GB
ROCm Mimari Kodu gfx1151

Strix Halo'nun Avantajlari

  • Devasa Bellek: NVIDIA RTX 4090'in 24 GB'ina karsi 96-112 GB GPU bellegi
  • Buyuk Model Destegi: 70B-128B parametreli modelleri yukleyebilme
  • Kompakt Form Faktor: Mini PC veya dizustu bilgisayarlarda bulunabilir
  • Tum Sistem Bellegi: CPU ve GPU ayni bellek havuzunu paylasiyor

Dezavantajlar

  • Sinirli Bant Genisligi: ~212 GB/s (kurumsal GPU'lara gore dusuk)
  • ROCm Destegi: gfx1151 icin destek hala gelisiyor
  • hipBLASLt Sorunu: Bazi optimizasyonlar henuz tam calisimiyor

Hizli Kurulum Adimlari

AMD Suruculerini Yukleyin

AMD'nin PyTorch destekli ozel suruculerini indirin. AMD Software: PyTorch on Windows Edition 7.1.1 veya daha yeni surumu gerektir.

Python Ortamini Hazirlayƒ±n

Python 3.12 kurun ve sanal ortam olusturun. Anaconda veya venv kullanabilirsiniz.

PyTorch ROCm Surumunu Kurun

AMD'nin resmi wheel dosyalarindan PyTorch kurun:

pip install --no-cache-dir https://repo.radeon.com/rocm/windows/rocm-rel-7.1.1/torch-2.9.0+rocmsdk20251116-cp312-cp312-win_amd64.whl

Gerekli Kutuphaneleri Kurun

PEFT, transformers ve diger gerekli kutuphaneleri pip ile kurun.

Kurulumu Dogrulayin

Python'da GPU'nun tanindigini kontrol edin:

import torch
print(f"PyTorch surumu: {torch.__version__}")
print(f"CUDA mevcut: {torch.cuda.is_available()}")
print(f"GPU sayisi: {torch.cuda.device_count()}")
if torch.cuda.is_available():
    print(f"GPU adi: {torch.cuda.get_device_name(0)}")

Bilinen Sinirlamalar

Kritik Sinirlamalar:
  • Fine-tuning destegi henuz resmi degil - on izleme asamasinda
  • Unsloth ve diger populer araclar calismaz (xformers bagimliligi)
  • BitsAndBytes'in Windows + AMD destegi sinirli
  • Flash Attention consumer GPU'larda calismaz
  • Bazi ROCm kutuphaneleri (hipBLASLt) tam optimize degil

Onerilen Cozumler

  • WSL2 Kullanin: Linux alt sistemiyle ROCm kullanmak daha stabil olabilir
  • Linux'a Gecis: Ciddi fine-tuning icin Linux kullanin
  • DirectML Inference: Sadece model calistirma icin DirectML kullanin
  • Kucuk Batch Size: Bellek hatalarinda batch boyutunu dusurun

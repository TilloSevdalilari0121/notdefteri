2. AMD Surucu Kurulumu

Fine-tuning icin standart Adrenalin suruculeri yetersizdir. AMD'nin PyTorch destekli ozel surucusunu kurmaniz gerekmektedir.

Mevcut Suruculeri Kaldir

Varolan AMD surucunuzu tamamen kaldirin. Bunu yapmanin en temiz yolu AMD Cleanup Utility kullanmaktir:

  1. AMD Cleanup Utility'yi indirin
  2. Bilgisayarinizi Guvenli Mod'da yeniden baslatin
  3. AMD Cleanup Utility'yi calistirin
  4. Islem bittiginde bilgisayari normal modda yeniden baslatin

PyTorch Surucusunu Indirin

AMD'nin resmi sayfasindan AMD Software: PyTorch on Windows Edition surucusunu indirin:

AMD Surucu Indirme Sayfasi

Surucu surumu 25.20.01.17 veya daha yeni olmalidir. (PyTorch Edition 7.1.1 icin)

Surucuyu Kurun

Indirdiginiz surucuyu calistirin ve kurulum sihirbazini takip edin. Kurulum tamamlandiginda bilgisayari yeniden baslatin.

Onemli: Surucu kurulumundan sonra sorun yasarsaniz, AMD Cleanup Utility ile temizleyip tekrar kurun.

3. Python Kurulumu

AMD'nin PyTorch surumu Python 3.12 gerektirmektedir.

Python 3.12 Indirin

Python'un resmi sitesinden 3.12 surumunu indirin: python.org/downloads

Kurulum Secenekleri

Kurulum sirasinda su secenekleri isaretleyin:

  • "Add Python to PATH" - Mutlaka isaretleyin
  • "Install for all users" - Onerilen
  • "Disable path length limit" - Kurulum sonunda yapilabilir

Kurulumu Dogrulayin

Komut istemini (CMD) acin ve su komutu calistirin:

python --version

Cikti Python 3.12.x olmalidir.

Alternatif: Anaconda/Miniconda

Python ortamini Anaconda ile de yonetebilirsiniz. Bu yontem birden fazla Python surumunu ve ortamini yonetmeyi kolaylastirir. 

# Miniconda kurulumundan sonra:
conda create --name rocm python=3.12
conda activate rocm

4. Sanal Ortam Olusturma

Sanal ortam, projelerinizin kutuphanelerini birbirinden izole eder. Bu sayede farkli projeler farkli kutuphane surumleri kullanabilir.

venv ile Sanal Ortam (Onerilen)

# Proje klasorunuze gidin veya yeni bir klasor olusturun
mkdir llm-finetuning
cd llm-finetuning

# Sanal ortam olusturun
python -m venv venv

# Sanal ortami aktif edin (Windows)
venv\Scripts\activate

# Aktif oldugunuzu dogrulayin - basinda (venv) yazmalı
(venv) C:\Users\...\llm-finetuning>

Conda ile Sanal Ortam

# Yeni conda ortami olusturun
conda create --name llm-finetune python=3.12

# Ortami aktif edin
conda activate llm-finetune

# Aktif oldugunuzu dogrulayin - basinda ortam adi yazmalı
(llm-finetune) C:\Users\...>
Ipucu: Her calisma oturumunda sanal ortami aktif etmeyi unutmayin. Aktif olmadan paket kurarsaniz, sistem Python'una yuklenir.

5. PyTorch ROCm Kurulumu

AMD'nin PyTorch surumunu resmi wheel dosyalarindan kuruyoruz. pip'in standart PyTorch paketini kurmayın.

pip'i Guncelleyin

python -m pip install --upgrade pip

PyTorch ve Ilgili Paketleri Kurun

Asagidaki komutlari tek tek calistirin (surumler degisebilir, AMD dokumantasyonunu kontrol edin): 

# PyTorch ana paketi
pip install --no-cache-dir https://repo.radeon.com/rocm/windows/rocm-rel-7.1.1/torch-2.9.0+rocmsdk20251116-cp312-cp312-win_amd64.whl

# TorchVision (goruntu islemleri icin)
pip install --no-cache-dir https://repo.radeon.com/rocm/windows/rocm-rel-7.1.1/torchvision-0.22.0+rocmsdk20251116-cp312-cp312-win_amd64.whl

# TorchAudio (ses islemleri icin - opsiyonel)
pip install --no-cache-dir https://repo.radeon.com/rocm/windows/rocm-rel-7.1.1/torchaudio-2.9.0+rocmsdk20251116-cp312-cp312-win_amd64.whl

Kurulumu Test Edin

Python'u acin ve su komutlari calistirin:

import torch

print(f"PyTorch surumu: {torch.__version__}")
print(f"ROCm surumu: {torch.version.hip}")
print(f"CUDA mevcut (AMD icin True olmali): {torch.cuda.is_available()}")
print(f"GPU sayisi: {torch.cuda.device_count()}")

if torch.cuda.is_available():
    print(f"GPU adi: {torch.cuda.get_device_name(0)}")
    print(f"GPU bellegi: {torch.cuda.get_device_properties(0).total_memory / 1024**3:.2f} GB")

Beklenen cikti:

PyTorch surumu: 2.9.0+rocmsdk20251116
ROCm surumu: 7.1.52802
CUDA mevcut (AMD icin True olmali): True
GPU sayisi: 1
GPU adi: AMD Radeon RX 7900 XTX
GPU bellegi: 24.00 GB
Hata Aliyorsaniz:
  • GPU adi gorunmuyorsa surucuyu kontrol edin
  • "CUDA not available" hatasi aliyorsaniz PyTorch ROCm surumunu dogru kurdugunuzdan emin olun
  • pip install sırasında hata alıyorsanız --no-cache-dir kullanın

7. Kurulum Dogrulama

Her seyin dogru kuruldugunu dogrulamak icin su scripti calistirin:

# dogrulama.py olarak kaydedin ve calistirin

import sys
print("=" * 50)
print("KURULUM DOGRULAMA")
print("=" * 50)

# Python surumu
print(f"\n[1] Python surumu: {sys.version}")

# PyTorch
try:
    import torch
    print(f"\n[2] PyTorch surumu: {torch.__version__}")
    print(f"    ROCm/HIP surumu: {torch.version.hip}")
    print(f"    CUDA mevcut: {torch.cuda.is_available()}")

    if torch.cuda.is_available():
        print(f"    GPU sayisi: {torch.cuda.device_count()}")
        for i in range(torch.cuda.device_count()):
            props = torch.cuda.get_device_properties(i)
            print(f"    GPU {i}: {props.name}")
            print(f"           Bellek: {props.total_memory / 1024**3:.2f} GB")

        # Basit GPU testi
        print("\n    GPU hesaplama testi...")
        x = torch.randn(1000, 1000, device="cuda")
        y = torch.matmul(x, x)
        print(f"    GPU hesaplama: BASARILI")
    else:
        print("    UYARI: GPU bulunamadi!")
except ImportError as e:
    print(f"\n[2] HATA: PyTorch kurulamadi - {e}")

# Transformers
try:
    import transformers
    print(f"\n[3] Transformers surumu: {transformers.__version__}")
except ImportError:
    print("\n[3] HATA: Transformers kurulamadi")

# PEFT
try:
    import peft
    print(f"\n[4] PEFT surumu: {peft.__version__}")
except ImportError:
    print("\n[4] HATA: PEFT kurulamadi")

# Accelerate
try:
    import accelerate
    print(f"\n[5] Accelerate surumu: {accelerate.__version__}")
except ImportError:
    print("\n[5] HATA: Accelerate kurulamadi")

# Datasets
try:
    import datasets
    print(f"\n[6] Datasets surumu: {datasets.__version__}")
except ImportError:
    print("\n[6] HATA: Datasets kurulamadi")

print("\n" + "=" * 50)
print("DOGRULAMA TAMAMLANDI")
print("=" * 50)

Scripti calistirmak icin:

python dogrulama.py
Basarili Kurulum: Tum satirlarda surum numaralari gorunuyorsa ve GPU hesaplama testi "BASARILI" yaziyorsa, kurulumunuz tamamdir!

8. Sorun Giderme

Sik Karsilasilan Hatalar

Hata: "torch.cuda.is_available()" False dönüyor

Neden: GPU surucusu veya PyTorch dogru kurulmamis.

Cozum:

  1. AMD PyTorch suruculerinin kurulu oldugundan emin olun
  2. pip ile kurdugunuz PyTorch'un ROCm surumu oldugundan emin olun
  3. Bilgisayari yeniden baslatin

Hata: "Out of Memory" (Bellek Yetersiz)

Neden: Model veya batch boyutu GPU bellegine sigmıyor.

Cozum:

  1. Batch size degerini dusurun (ornegin 4'ten 2'ye veya 1'e)
  2. Gradient checkpointing'i aktif edin
  3. Daha kucuk bir model kullanin
  4. LoRA rank degerini dusurun (ornegin 16'dan 8'e)

Hata: "No module named 'torch'"

Neden: Sanal ortam aktif degil veya PyTorch kurulmamis.

Cozum:

  1. Sanal ortami aktif edin: venv\Scripts\activate
  2. PyTorch'u yeniden kurun

Hata: "hipErrorNoBinaryForGpu"

Neden: GPU mimariniz desteklenmiyor veya kernel bulunamadi.

Cozum:

  1. GPU'nuzun desteklenen listede oldugundan emin olun
  2. En son suruculeri kurun
  3. HSA_OVERRIDE_GFX_VERSION ortam degiskenini ayarlayin (ileri duzey)

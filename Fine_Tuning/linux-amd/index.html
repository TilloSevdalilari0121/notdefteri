Hizli ROCm Kurulumu (Ubuntu)

Ubuntu 22.04 veya 24.04 icin onerilen kurulum adimlari:

Sistemi Guncelleyin

sudo apt update && sudo apt upgrade -y

AMDGPU Surucusunu Kurun

# Ubuntu 24.04 icin
wget https://repo.radeon.com/amdgpu-install/6.2.2/ubuntu/noble/amdgpu-install_6.2.60202-1_all.deb
sudo apt install ./amdgpu-install_6.2.60202-1_all.deb
sudo amdgpu-install -y --usecase=rocm

Kullaniciyi Gruplara Ekleyin

sudo usermod -a -G render,video $USER

Sistemi Yeniden Baslatin

sudo reboot

Kurulumu Dogrulayin

rocminfo
rocm-smi

Docker ile Hizli Baslangic

Kurulum karmasikligindan kacinmak icin AMD'nin resmi Docker imajlarini kullanabilirsiniz.

PyTorch ROCm Docker

# ROCm PyTorch Docker'i calistir
docker run -it --rm \
    --device=/dev/kfd \
    --device=/dev/dri \
    --group-add=video \
    --ipc=host \
    --shm-size 8G \
    -v $(pwd):/workspace \
    rocm/pytorch:latest

Llama Factory Docker

# Container icinde Llama Factory kur
git clone https://github.com/hiyouga/LLaMA-Factory.git
cd LLaMA-Factory
pip install -e ".[torch,metrics]"

# Fine-tuning baslat
llamafactory-cli train examples/train_lora/llama3_lora_sft.yaml
Avantaj: Docker ile tum bagimliliklarin uyumlu surumlerini hazƒ±r olarak alirsiniz. Sistem Python'unuza dokunmadan calisabilirsiniz.

Hizli Fine-Tuning Ornegi

Llama Factory ile basit bir fine-tuning ornegi:

# 1. Ortami hazirla (Docker icinde veya disinda)
pip install transformers datasets peft accelerate trl

# 2. Llama Factory'yi kur
git clone https://github.com/hiyouga/LLaMA-Factory.git
cd LLaMA-Factory
pip install -e ".[torch,metrics]"

# 3. Web UI'yi baslat (kolay kullanim icin)
llamafactory-cli webui

# VEYA komut satirindan:
# 4. Yapilandirma dosyasini duzenle ve calistir
llamafactory-cli train examples/train_lora/llama3_lora_sft.yaml

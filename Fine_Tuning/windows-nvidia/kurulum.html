Kurulum Adimlari

NVIDIA Surucusunu Guncelleyin

NVIDIA Surucu Indirme sayfasindan GPU'nuz icin en son surucuyu indirin ve kurun.

Surucu surumu 535+ olmalidir (CUDA 12+ destegi icin).

Python Kurun

python.org'dan Python 3.10 veya 3.11 indirin. Kurulum sirasinda "Add Python to PATH" secenegini isaretleyin. 

# Kurulumu dogrula
python --version
# Python 3.10.x veya 3.11.x olmali

Visual Studio Build Tools (Opsiyonel)

Bazi paketler C++ derleyicisi gerektirir. Visual Studio Community'yi "Desktop development with C++" secenegiyle kurun.

Visual Studio Build Tools

mkdir llm-finetuning

venv\Scripts\activate

Kurulumu Dogrulayin

python -c "import torch; print(f'PyTorch: {torch.__version__}'); print(f'CUDA: {torch.cuda.is_available()}'); print(f'GPU: {torch.cuda.get_device_name(0) if torch.cuda.is_available() else None}')"

PyTorch: 2.5.1+cu126
CUDA: True

Unsloth Kurulumu

Onerilen: Unsloth, NVIDIA GPU'larda fine-tuning icin en hizli ve en bellek verimli aractir. Kurulumu cok basittir.

Temel Kurulum

# Unsloth'u kur (PyTorch kurulu olmali)
pip install unsloth

# Ek bagimliliklar
pip install transformers datasets accelerate peft trl

Kurulumu Dogrula

python -c "from unsloth import FastLanguageModel; print('Unsloth basariyla yuklendi!')"

Windows Icin Ek Notlar

  • Triton kurulumu icin Windows fork gerekebilir
  • Bazi hatalarda dataset_num_proc=1 ayarlayin
  • Sorun yasarsanız WSL2 veya Docker deneyin

Alternatif: WSL2 ile Linux Kullanimi

Bazi araclar (Axolotl, Llama Factory) Windows'ta yerel olarak calismaz. WSL2 ile Linux ortami kullanabilirsiniz. 

# PowerShell'i yonetici olarak acin
wsl --install -d Ubuntu-22.04

# WSL2 kullanildigindan emin olun
wsl --set-version Ubuntu-22.04 2

# Ubuntu'ya giris
wsl

WSL2'de NVIDIA GPU otomatik olarak kullanilabilir (CUDA on WSL).

Linux + NVIDIA Rehberine Git

Sorun Giderme

torch.cuda.is_available() False dönuyor

  • NVIDIA surucusunun kurulu oldugundan emin olun
  • PyTorch'un CUDA surumunu kurdugunuzdan emin olun (CPU degil)
  • GPU'nun CUDA 7.0+ desteklediginden emin olun

Out of Memory (OOM) Hatasi

  • Batch size'i dusurun
  • QLoRA (4-bit) kullanin
  • Daha kucuk model secin
  • Gradient checkpointing aktif edin
